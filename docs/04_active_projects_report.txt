PROJECT 7: INTERNAL — AI INFRASTRUCTURE EVALUATION
--------------------------------------------------------------------------------
Customer: Internal (Nexus Technology Solutions)
Project Sponsor: Rachel Nguyen, VP of Engineering
Project Manager: Maria Santos (maria.santos@nexustech.io)
Technical Lead: Jake Thompson (jake.thompson@nexustech.io)

Budget: $45,000 (approved, from operations budget)
Start Date: December 2025
Target Completion: March 31, 2026
Current Status: IN PROGRESS (Green)

Project Overview:
Nexus is evaluating AI infrastructure options for two internal use cases:
(1) AI-assisted NOC alert triage and incident summarization, and (2) internal
knowledge base chatbot using RAG over company documentation.

Evaluation criteria:
- Self-hosted LLM options (vLLM on H100 GPU nodes already owned)
- Managed API options (Azure OpenAI, Anthropic Claude, OpenAI)
- Open source models: Llama 3.1 70B, Mistral Large, Mixtral 8x22B
- Cost per query, latency, quality, data privacy implications

Current findings:
- Self-hosted vLLM on H100 nodes is technically viable and provides best
  data privacy (no customer data leaves premises)
- Azure OpenAI GPT-4o provides best quality for document summarization
  but cost at scale is $0.010/1K output tokens
- Anthropic Claude Sonnet via API scored highest on internal knowledge base
  Q&A accuracy in blind evaluation (87% accuracy vs 79% for GPT-4o)
- Local Llama 3.1 70B on H100 is 6x slower than API options but cost is
  effectively $0 at current scale

Recommendation expected: February 15, 2026

Budget Status: $12,000 spent (discovery, GPU node evaluation, API costs)
